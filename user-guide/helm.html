Installing Ambassador with Helm

Helm is a package manager for Kubernetes. Ambassador is available as a Helm chart if you use Helm for package management. To install with Helm:

  1. Add the Ambassador Helm repository:

    helm repo add datawire https://www.getambassador.io

  2. Install the Ambassador Chart:

    helm upgrade --install --wait my-release datawire/ambassador

  3. Jump to step 3 of the Ambassador tutorial to create your first route.

Configuring the chart

For details on how to configure the chart, see the official chart documentation here: https://github.com/datawire/ambassador/tree/stable/helm/ambassador.
